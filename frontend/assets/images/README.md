# assets/images/

Place static image assets here (PNG, JPG, WebP).

## Recommended files

| File | Usage |
|---|---|
| `og-image.png` | Open Graph / social share preview (1200×630 px) |
| `logo.png` | App logo for in-app use |
| `empty_state.png` | Illustration for empty projects list |
| `onboarding_1.png` | Onboarding slide 1 (optional) |
| `onboarding_2.png` | Onboarding slide 2 (optional) |

All images in this folder are automatically bundled by Flutter.
Reference them with: `Image.asset('assets/images/your_file.png')`
