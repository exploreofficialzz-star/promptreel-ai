# assets/icons/

Place custom SVG or PNG icon files here.

## Usage

```dart
// SVG icons (flutter_svg)
SvgPicture.asset('assets/icons/my_icon.svg', width: 24)

// PNG icons
Image.asset('assets/icons/my_icon.png', width: 24)
```

The project uses `iconsax_flutter` for most UI icons.
Only place custom brand/product-specific icons here.
