# assets/animations/

Place Lottie animation files here (.json format).

## Recommended files

| File | Usage |
|---|---|
| `loading.json` | Generation loading animation |
| `success.json` | Success/completion animation |
| `empty.json` | Empty state animation |
| `ai_thinking.json` | AI processing indicator |

Source free Lottie files from: https://lottiefiles.com

Reference in code:
```dart
Lottie.asset('assets/animations/loading.json', width: 120)
```
